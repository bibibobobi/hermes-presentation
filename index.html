PITFALLS
踩坑紀錄
❌ Claude Pro 無法直接用
借到 session token 但會失效，造成 bot 斷線。

正解：改用 OpenRouter API key，穩定不斷線。
❌ 免費 model 太慢
Nemotron 跑了整整 3 分鐘沒有回應，完全卡死。

正解：換成 Gemini 3 Flash，秒回。
❌ Guardrail 擋掉免費 model
免費端點被安全策略攔截，無法正常使用。

正解：關掉「Free endpoints that may train on request data」
❌ 沒接搜尋工具，他假裝在搜尋 😄
沒設定 Tavily / SearXNG / Firecrawl，HermesBot 會說「正在執行搜尋動作...」然後用訓練資料回答——根本沒有真的上網。

正解：設定任一搜尋後端，確認 setup 畫面顯示 ✅ Web Search & Extract。推薦：Tavily（免費 1,000 credits/月）

01 · VPS 隔離
部署在 VPS，不放本機
Hermes 有完整本機存取權（檔案、終端、.env）。部署在獨立 server，操作範圍限制在那台機器，不影響個人電腦。
02 · Prompt Injection
開啟 OpenRouter Guardrail
瀏覽網頁時，頁面可能藏有惡意指令試圖操控 LLM。建議在 OpenRouter 後台啟用 Guardrail 作為防護層。
03 · Skill 審查
定期 review ~/.hermes/skills/
把每次新增的 skill 當作 code review PR 一樣看待。確認它做什麼、存取哪些資源，才能真正掌握 agent 的行為邊界。
